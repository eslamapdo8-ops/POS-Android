# نظام نقاط البيع لمتجر الملابس (POS)

نظام نقاط بيع مبسّط لمحلات الملابس، يعمل على الأجهزة اللوحية بأندرويد.

## المميزات

- **شاشة البيع (POS)**: شبكة منتجات مع سلة شراء ودفع نقدي/بطاقة
- **إدارة المنتجات**: إضافة وتعديل وحذف المنتجات
- **مسح الباركود**: كاميرا مع ML Kit لقراءة الباركود
- **طباعة الإيصالات**: دعم طابعات بلوتوث ESC/POS
- **نسخ احتياطي**: تصدير البيانات إلى Google Sheets (اختياري)
- **واجهة عربية**: دعم كامل للغة العربية RTL

## المتطلبات

- أندرويد 7.0 (API 24) أو أحدث
- جهاز لوحي أو هاتف بأندرويد
- كاميرا (لمسح الباركود - اختياري)
- بلوتوث (للطابعة - اختياري)

## التقنيات

| التقنية | الاستخدام |
|---------|-----------|
| Kotlin | لغة البرمجة |
| Room | قاعدة البيانات المحلية |
| CameraX + ML Kit | مسح الباركود |
| ESC/POS | الطباعة الحرارية |
| Material Design 3 | واجهة المستخدم |

## البناء والتشغيل

### عبر GitHub Actions
يتم بناء APK تلقائياً عند كل دفعة إلى فرع `main`. يمكن تحميل APK من قسم Artifacts في صفحة الـ workflow.

### محلياً باستخدام Android Studio
1. استنساخ المستودع
2. فتح المشروع في Android Studio
3. مزامنة Gradle
4. تشغيل التطبيق على جهاز أو محاكي

## هيكل المشروع

```
app/src/main/java/com/pos/clothingstore/
├── POSApplication.kt          # فئة التطبيق + بيانات تجريبية
├── data/
│   ├── local/
│   │   ├── AppDatabase.kt     # قاعدة البيانات (Room)
│   │   ├── entity/            # كيانات الجداول
│   │   └── dao/               # عمليات قاعدة البيانات
├── ui/
│   ├── MainActivity.kt        # النشاط الرئيسي
│   ├── pos/                   # شاشة البيع
│   ├── products/              # إدارة المنتجات
│   ├── scanner/               # مسح الباركود
│   └── backup/                # النسخ الاحتياطي
└── printer/                   # الطباعة بالبلوتوث
```

## قاعدة البيانات

| الجدول | الوصف |
|--------|-------|
| `products` | المنتجات (باركود، اسم، سعر، كمية، مورد) |
| `sales` | فواتير البيع (تاريخ، إجمالي، طريقة دفع) |
| `sale_items` | بنود الفاتورة (منتج، كمية، سعر) |
| `users` | المستخدمون (إيميل، دور) |
